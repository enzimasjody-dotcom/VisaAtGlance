# Frontend 작업 규칙

이 frontend는 Next.js 16 App Router 기반이다. Next.js 관련 구현을 바꿀 때는 현재 설치된 버전의 동작을 우선 확인한다.

참고 기준:

- 기존 기억보다 `frontend/node_modules/next/dist/docs/`의 현재 문서를 우선한다.
- deprecation warning이 나오면 무시하지 않는다.
- dashboard UI는 marketing page보다 public data preview를 우선한다.
- Tableau 같은 analytics platform 구조는 참고만 하고 복제하지 않는다.
