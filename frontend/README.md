# VisaAtGlance Frontend

Next.js frontend for VisaAtGlance.

## 실행

루트에서 실행:

```bash
make visa-frontend-dev
```

또는 frontend 폴더에서 직접 실행:

```bash
npm install
npm run dev
```

기본 주소:

```text
http://localhost:3000
```

## 확인

```bash
make visa-frontend-check
```

위 명령은 lint와 production build를 실행한다.
